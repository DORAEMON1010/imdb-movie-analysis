# IMDB Movie Dataset Analysis

This project performs a comprehensive exploratory data analysis (EDA) of the IMDB Movie Dataset using Python and various data visualization tools. The aim is to discover insights about movie trends, ratings, revenues, popular genres, and more.

## Key Analysis Performed:

1. Displaying top and bottom records of the dataset
2. Dataset shape and structure inspection
3. Null value detection and removal
4. Checking for duplicate records
5. Statistical summary of numeric features
6. Finding longest runtime movies
7. Identifying year-wise trends in:
   - Votes
   - Revenues
8. Director-wise average ratings
9. Most popular and highest-grossing movies
10. Year-wise average movie ratings
11. Correlation between ratings and revenue
12. Categorizing movies by rating levels (Excellent, Good, Average)
13. Genre-based filtering and action movie count

## Tools & Libraries Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Dataset

You can download the dataset [here](https://www.kaggle.com/datasets/PromptCloudHQ/imdb-data) or use the file provided in the repository.

## Conclusion

This analysis offers insights into how different movie attributes relate to success indicators like rating and revenue, and highlights trends across years and genres.
